## 2. 伤害类型

```csharp
public enum EDamageType
{
    Physical,    // 物理伤害
    Magic,      // 魔法伤害
    True,       // 真实伤害（无视护甲魔抗）
    Pure,       // 纯粹伤害（无视一切减伤）
}

public readonly struct DamageInfo
{
    public int AttackerId;
    public int TargetId;
    public EDamageType DamageType;
    public int Value;
    public EDamageTag Tag;  // 区分来源（技能、Buff、反弹等）
}
```

## 3. 护甲减伤公式

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           护甲减伤公式                                     │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   减伤比例 = 护甲值 / (护甲值 + 100)                                 │
│                                                                             │
│   示例：                                                                  │
│   • 护甲 0 → 减伤 0%                                                    │
│   • 护甲 50 → 减伤 33%                                                  │
│   • 护甲 100 → 减伤 50%                                                 │
│   • 护甲 200 → 减伤 67%                                                 │
│   • 护甲 300 → 减伤 75%                                                 │
│                                                                             │
│   最终伤害 = 基础伤害 × (1 - 减伤比例)                               │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 4. 伤害流程

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           伤害流程                                         │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   1. 获取攻击方攻击力                                                    │
│      attackerAttack = GetFinalAttribute(AttackerId, Attack)              │
│                                                                             │
│   2. 计算基础伤害                                                        │
│      baseDamage = skillDamage + attackerAttack × coefficient              │
│                                                                             │
│   3. 获取防御方属性                                                       │
│      defense = GetFinalAttribute(TargetId, Defense/MagicDefense)          │
│                                                                             │
│   4. 计算减伤比例                                                        │
│      reduction = defense / (defense + 100)                               │
│                                                                             │
│   5. 计算最终伤害                                                        │
│      finalDamage = baseDamage × (1 - reduction)                         │
│                                                                             │
│   6. 扣除生命值                                                          │
│      target.Hp -= finalDamage                                            │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
