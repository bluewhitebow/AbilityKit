## 1. Buff 概述

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           Buff 概述                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   Buff = 增益效果（对自己有利）                                          │
│   Debuff = 减益效果（对目标不利）                                        │
│                                                                             │
│   常见类型：                                                              │
│   ├── 属性增益/减益：攻击力+10%、移动速度-30%                           │
│   ├── 护盾：吸收伤害                                                     │
│   ├── 无敌：免疫所有伤害                                                │
│   ├── 沉默：无法释放技能                                                │
│   └── 眩晕：无法移动和攻击                                              │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 2. Buff 配置

```csharp
public sealed class BuffConfig
{
    public int BuffId;
    public string Name;
    public EBuffType Type;
    public float Duration;
    public int MaxStacks;
    public EBuffStackingRule StackingRule;
    public int Priority;
    public bool Refreshable;
    public BuffStageConfig Stage;
}

public enum EBuffType
{
    Buff,       // 增益
    Debuff,     // 减益
    Shield,     // 护盾
    Invincible, // 无敌
    Silence,    // 沉默
    Stun,      // 眩晕
}
```

## 4. Buff 叠加规则

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           Buff 叠加规则                                   │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   None (不叠加)                                                           │
│   └── 新 Buff 直接替换旧 Buff                                            │
│                                                                             │
│   Refresh (刷新)                                                           │
│   └── 新 Buff 刷新旧 Buff 的持续时间，但不增加层数                        │
│                                                                             │
│   AddStack (增加层数)                                                     │
│   └── 新 Buff 增加层数，持续时间刷新                                      │
│                                                                             │
│   Independent (独立)                                                       │
│   └── 新 Buff 作为独立实例存在                                            │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
