## 2. 注册方式

```csharp
// 单例注册
container.RegisterSingleton<IEventDispatcher, EventDispatcher>();
container.RegisterSingleton<ILogger, ConsoleLogger>();

// 瞬态注册（每次解析都创建新实例）
container.RegisterTransient<ICommand, MoveCommand>();

// 实例注册
container.RegisterSingleton<IDamageService>(damageServiceInstance);
```

---

## 3. 解析方式

```csharp
// 直接解析
var executor = container.Resolve<SkillExecutor>();

// 安全解析
if (container.TryResolve<IDamageService>(out var damageService))
{
    damageService.ApplyDamage(...);
}

// 解析所有
var handlers = container.ResolveAll<IEventHandler>();
```

## 4. 生命周期

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           生命周期类型                                     │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   Singleton (单例)                                                       │
│   └── 在容器范围内只创建一次，应用期间一直存在                      │
│                                                                             │
│   Transient (瞬态)                                                       │
│   └── 每次解析都创建新实例，用完即弃                                  │
│                                                                             │
│   Scoped (作用域)                                                       │
│   └── 在作用域内单例，如数据库连接                                  │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
