## 2. 接口定义

```csharp
public interface IObjectPool<T> where T : class
{
    /// <summary>获取对象</summary>
    T Rent();

    /// <summary>归还对象</summary>
    void Return(T obj);

    /// <summary>预热池子</summary>
    void Warmup(int count);
}
```

---

## 3. 使用示例

```csharp
// 投射物池
var projectilePool = new ObjectPool<ProjectileEntity>(
    factory: () => new ProjectileEntity(),
    onRent: p => p.Reset(),
    onReturn: p => p.Clear()
);

// 使用
var projectile = projectilePool.Rent();
projectile.Launch(caster, target, config);
projectilePool.Return(projectile);

// Buff 实例池
var buffPool = new ObjectPool<BuffInstance>(
    factory: () => new BuffInstance()
);

var buff = buffPool.Rent();
buff.Initialize(actorId, config);
buffPool.Return(buff);
```

## 4. 适用场景

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           适用场景                                         │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   ✅ 适合使用对象池                                                     │
│   ├── 投射物（数量多，生命周期短）                                       │
│   ├── Buff 实例（频繁创建销毁）                                          │
│   ├── 伤害数字（VFX）                                                   │
│   ├── 事件对象（发布订阅）                                               │
│   └── 命令对象（输入处理）                                               │
│                                                                             │
│   ❌ 不适合使用对象池                                                   │
│   ├── 生命周期长的对象                                                   │
│   └── 低频创建的对象                                                     │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
