## 1. 术语体系概览

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           AbilityKit 术语体系                               │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   世界层                                                                  │
│   ├── World        ─── 逻辑世界容器                                       │
│   ├── Entity      ─── 战斗单位实体                                        │
│   ├── Component   ─── 实体数据组件                                        │
│   └── System      ─── 逻辑处理系统                                        │
│                                                                             │
│   帧同步层                                                                │
│   ├── Frame       ─── 游戏逻辑帧                                          │
│   ├── Input       ─── 玩家输入命令                                        │
│   ├── Snapshot    ─── 世界状态快照                                        │
│   └── Rollback    ─── 状态回滚                                            │
│                                                                             │
│   技能层                                                                  │
│   ├── Skill       ─── 技能                                                │
│   ├── Trigger     ─── 触发器                                              │

│   └── Buff        ─── 增益/减益状态                                        │
│                                                                             │
│   网络层                                                                  │
│   ├── Session     ─── 战斗会话                                            │

│   └── Adapter     ─── 同步适配器                                          │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 2. ECS 概念映射

### 2.1 基础概念

| 概念 | 英文 | 说明 | 类比 |
|------|------|------|------|
| **实体** | Entity | 唯一标识符，不包含数据 | 身份证号 |
| **组件** | Component | 纯数据，贴在实体上 | 档案卡片 |
| **系统** | System | 纯逻辑，处理组件 | 档案管理员 |

### 2.2 AbilityKit 中的映射

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           ECS 在 AbilityKit 中的映射                       │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   Entity (实体)                                                            │
│   ┌─────────────────────────────────────────────────────────────────────┐ │
│   │  ActorEntity                                                         │ │
│   │                                                                       │ │
│   │  • 唯一 ID（ActorId）                                               │ │
│   │  • 不包含任何数据                                                    │ │
│   │  • 只是一个"钩子"，用于关联组件                                      │ │
│   └─────────────────────────────────────────────────────────────────────┘ │
│                                                                             │
│   Component (组件)                                                          │
│   ┌─────────────────────────────────────────────────────────────────────┐ │
│   │  贴在 ActorEntity 上的数据                                          │ │
│   │                                                                       │ │
│   │  • ActorTransformComponent  ─── 位置、旋转                          │ │
│   │  • ActorAttributeComponent  ─── 生命值、攻击力等                    │ │
│   │  • ActorSkillComponent     ─── 技能列表、冷却                        │ │
│   │  • ActorBuffComponent      ─── Buff 列表                            │ │
│   └─────────────────────────────────────────────────────────────────────┘ │
│                                                                             │
│   System (系统)                                                            │
│   ┌─────────────────────────────────────────────────────────────────────┐ │
│   │  处理特定类型组件的逻辑                                              │ │
│   │                                                                       │ │
│   │  • MobaMotionSystem      ─── 处理移动                               │ │
│   │  • SkillExecutor         ─── 处理技能释放                            │ │
│   │  • DamageSystem          ─── 处理伤害结算                            │ │
│   │  • BuffTickSystem        ─── 处理 Buff 周期效果                     │ │
│   └─────────────────────────────────────────────────────────────────────┘ │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 3. 帧同步概念

### 3.1 核心概念定义

| 概念 | 英文 | 说明 |
|------|------|------|
| **帧** | Frame | 游戏逻辑 Tick 的基本单位 |
| **输入** | Input | 玩家操作（移动、技能） |
| **快照** | Snapshot | 世界状态的序列化 |
| **回滚** | Rollback | 状态恢复到之前的某个帧 |

### 3.2 帧 (Frame)

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                              帧 (Frame)                                   │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   帧是游戏逻辑 Tick 的基本单位                                             │
│                                                                             │
│   ┌─────────┐   ┌─────────┐   ┌─────────┐   ┌─────────┐               │
│   │ Frame 1 │ → │ Frame 2 │ → │ Frame 3 │ → │ Frame 4 │ → ...        │
│   └─────────┘   └─────────┘   └─────────┘   └─────────┘               │
│       │             │             │             │                         │
│       ▼             ▼             ▼             ▼                         │
│   收集输入      执行逻辑      广播状态      推进帧号                      │
│                                                                             │
│   帧率配置：30 FPS 或 60 FPS                                              │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 3.3 输入 (Input)

```csharp
// 输入命令结构
public struct PlayerInputCommand
{
    public PlayerId PlayerId;      // 谁发的
    public FrameIndex Frame;       // 哪一帧
    public EInputType Type;        // 输入类型
    public byte[] Payload;         // 输入数据
}

// 输入类型
public enum EInputType
{
    None,
    Move,      // 移动
    Skill,     // 技能
    Stop,      // 停止
    Attack,    // 攻击
}
```

## 4. 技能系统概念

### 4.1 技能流程概念

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           技能流程概念                                     │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   施法请求                                                                │
│   └── SkillCastRequest                                                    │
│       ├── CasterId (谁放的)                                               │
│       ├── SkillId (放的什么技能)                                          │
│       └── TargetId / TargetPosition (目标)                                │
│                                                                             │
│   技能验证                                                                │
│   └── Validation                                                         │
│       ├── CooldownCheck (冷却检查)                                        │
│       ├── ManaCheck (魔法值检查)                                          │
│       ├── RangeCheck (距离检查)                                           │
│       └── TargetCheck (目标检查)                                           │
│                                                                             │
│   技能执行                                                                │
│   └── Execution                                                          │
│       ├── PreCast (前摇)                                                  │
│       ├── Cast (施法)                                                     │
│       └── PostCast (后摇)                                                 │
│                                                                             │
│   效果触发                                                                │
│   └── Trigger                                                            │
│       ├── Damage (伤害)                                                   │
│       ├── Buff (Buff)                                                     │
│       └── Projectile (投射物)                                             │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 4.2 触发器概念

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           触发器 (Trigger)                                  │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   触发器 = 触发源 + 条件 + 动作                                           │
│                                                                             │
│   ┌─────────────────────────────────────────────────────────────────────┐ │
│   │  触发源 (Trigger Source)                                           │ │
│   │                                                                       │ │
│   │  • OnHit         ─── 受击时                                         │ │
│   │  • OnSkillCast   ─── 技能释放时                                     │ │
│   │  • OnBuffApply   ─── Buff 应用时                                    │ │
│   │  • OnDeath       ─── 死亡时                                         │ │
│   │  • OnTimeElapsed  ─── 时间流逝                                       │ │
│   └─────────────────────────────────────────────────────────────────────┘ │
│                                                                             │
│   ┌─────────────────────────────────────────────────────────────────────┐ │
│   │  条件 (Condition)                                                  │ │
│   │                                                                       │ │
│   │  • TargetHasTag("CanBeDamaged")  ─── 目标有某标签                 │ │
│   │  • CasterHpBelow(0.3)         ─── 施法者血量低于 30%             │ │
│   │  • RandomChance(0.5)          ─── 50% 概率                        │ │
│   └─────────────────────────────────────────────────────────────────────┘ │
│                                                                             │
│   ┌─────────────────────────────────────────────────────────────────────┐ │
│   │  动作 (Executable)                                                  │ │
│   │                                                                       │ │
│   │  • ApplyDamage(100)     ─── 造成 100 点伤害                        │ │
│   │  • ApplyBuff("Slow")    ─── 施加"减速"Buff                        │ │
│   │  • SpawnProjectile(...) ─── 生成投射物                             │ │
│   └─────────────────────────────────────────────────────────────────────┘ │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 5. 会话与网络概念

### 5.1 会话生命周期

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                          会话生命周期                                       │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐         │
│   │   Idle   │ → │ Starting │ → │  Running │ → │ Stopping │ → Idle  │
│   └──────────┘    └──────────┘    └──────────┘    └──────────┘         │
│       │               │               │               │                   │
│       │               ▼               ▼               ▼                   │
│       │         ┌──────────┐    ┌──────────┐    ┌──────────┐            │
│       │         │Creating..│    │ InBattle │    │Destroying│            │
│       │         │  World   │    │  战斗中   │    │  World   │            │
│       │         └──────────┘    └──────────┘    └──────────┘            │
│       │                                                                         │
│       ◀─────────────────── Session.End() ───────────────────────         │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 5.2 同步适配器

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                          同步适配器 (SyncAdapter)                         │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   同步适配器封装了不同的网络同步策略                                        │
│                                                                             │
│   ┌─────────────────────────────────────────────────────────────────────┐ │
│   │  LocalSyncAdapter  ─── 本地帧同步                                   │ │
│   │                                                                       │ │
│   │  • 适用于单机游戏                                                   │ │
│   │  • 适用于局域网对战                                                 │ │
│   │  • 不需要网络传输                                                   │ │
│   └─────────────────────────────────────────────────────────────────────┘ │
│                                                                             │
│   ┌─────────────────────────────────────────────────────────────────────┐ │
│   │  RemoteSyncAdapter  ─── 远程同步（服务器权威）                      │ │
│   │                                                                       │ │
│   │  • 适用于网络游戏                                                   │ │
│   │  • 服务器是权威节点                                                 │ │
│   │  • 客户端只负责渲染                                                 │ │
│   └─────────────────────────────────────────────────────────────────────┘ │
│                                                                             │
│   ┌─────────────────────────────────────────────────────────────────────┐ │
│   │  HybridSyncAdapter  ─── 混合同步                                    │ │
│   │                                                                       │ │
│   │  • 适用于复杂战斗场景                                               │ │
│   │  • 支持客户端预测                                                   │ │
│   │  • 支持服务器校验                                                   │ │
│   └─────────────────────────────────────────────────────────────────────┘ │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
