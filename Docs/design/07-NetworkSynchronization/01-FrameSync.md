## 1. 帧同步基础

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           帧同步核心原理                                  │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   帧同步 = 所有客户端运行相同的逻辑，输入作为唯一的"种子"            │
│                                                                             │
│   ┌─────────────────────────────────────────────────────────────────────┐ │
│   │                                                                       │ │
│   │  帧 1: 收集所有玩家输入 → 执行逻辑 → 广播状态                     │ │
│   │  帧 2: 收集所有玩家输入 → 执行逻辑 → 广播状态                     │ │
│   │  帧 3: 收集所有玩家输入 → 执行逻辑 → 广播状态                     │ │
│   │  ...                                                                 │ │
│   │                                                                       │ │
│   │  输入相同 + 逻辑相同 + 初始状态相同 = 结果相同（确定性）            │ │
│   └─────────────────────────────────────────────────────────────────────┘ │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 2. 帧循环流程

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           帧循环时序                                      │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   LocalSyncAdapter.Tick()                                                  │
│                                                                             │
│   1. WaitForInputs() ─── 等待所有玩家输入到达                           │
│                                                                             │
│   2. ProcessLogicFrame()                                                  │
│      ├── 收集输入                                                        │
│      ├── world.Tick(deltaTime) ─── 执行所有系统                         │
│      └── 推进帧号                                                        │
│                                                                             │
│   3. PublishSnapshot() ─── 发布快照到视图层                              │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 3. 输入管理

```csharp
public readonly struct PlayerInputCommand
{
    public readonly PlayerId PlayerId;    // 谁发的
    public readonly FrameIndex Frame;    // 哪一帧
    public readonly EInputType Type;    // 输入类型
    public readonly byte[] Payload;     // 输入数据
}

public enum EInputType
{
    None,
    Move,      // 移动
    Skill,     // 技能
    Stop,      // 停止
}
```

## 4. 确定性保证

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           确定性要求                                      │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   ✅ 必须保证的确定性                                                    │
│   ├── 浮点运算使用确定性实现                                          │
│   ├── 随机数使用确定性 RNG（种子固定）                               │
│   ├── 实体遍历顺序固定                                                │
│   └── 浮点数比较使用 epsilon                                          │
│                                                                             │
│   ❌ 禁止使用                                                            │
│   ├── DateTime.Now                                                    │
│   ├── Guid.NewGuid()                                                  │
│   └── 不确定的排序                                                     │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
