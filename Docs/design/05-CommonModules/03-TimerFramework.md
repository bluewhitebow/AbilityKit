## 2. 定时器接口

```csharp
public interface ITimer
{
    /// <summary>延迟执行</summary>
    ITimerHandle Delay(float seconds, Action callback);

    /// <summary>周期执行</summary>
    ITimerHandle Loop(float interval, Action callback);

    /// <summary>帧定时器</summary>
    ITimerHandle FrameDelay(int frames, Action callback);

    /// <summary>取消</summary>
    void Cancel(ITimerHandle handle);
}

public interface ITimerHandle
{
    bool IsValid { get; }
    void Cancel();
}
```

## 3. 使用示例

```csharp
// 延迟执行
_timer.Delay(2.0f, () =>
{
    // 2 秒后执行
    skillExecutor.CastSkill(casterId, skillId);
});

// 周期执行
var handle = _timer.Loop(1.0f, () =>
{
    // 每秒执行
    ApplyDotDamage(targetId, 10);
});

// 帧定时器
_timer.FrameDelay(30, () =>
{
    // 30 帧后执行
    RemoveBuff(targetId, buffId);
});

// 取消
handle.Cancel();
```

## 4. 注意事项

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           注意事项                                         │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   ⚠️ 帧同步环境下的定时器                                                │
│   ├── 使用帧定时器而非真实时间                                          │
│   ├── 避免使用 DateTime.Now                                              │
│   └── 周期任务需要考虑帧率                                               │
│                                                                             │
│   ⚠️ 内存管理                                                            │
│   ├── 取消后及时释放句柄                                                │
│   └── 避免在定时器回调中创建长生命周期对象                              │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
