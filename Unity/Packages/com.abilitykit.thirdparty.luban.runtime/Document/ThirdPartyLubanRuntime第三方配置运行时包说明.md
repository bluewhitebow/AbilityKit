# Ability-Kit ThirdParty Luban Runtime 第三方配置运行时包说明

> **阅读对象**：需要维护 Luban 配置加载运行时的开发者。
>
> **文档目标**：说明该包作为 Luban 第三方运行时依赖的边界。

---

## 一、包定位

Luban Runtime 用于配置表加载和反序列化。该包保存第三方 Luban 运行时相关代码或程序集，为 Demo Moba 和真实项目的配置读取提供基础能力。

---

## 二、边界说明

负责：

- 提供 Luban 生成配置的运行时读取支持。
- 支撑配置 DTO/表数据加载流程。

不负责：

- 不定义 AbilityKit 业务配置。
- 不生成配置代码。
- 不负责 Unity 编辑器配置资产导出。

---

## 三、维护建议

- Luban 生成器版本和 runtime 版本要匹配。
- 项目配置 DTO 和导出工具应放在 demo/editor 或业务包中。
- 保留上游授权和版本信息。
