## 2. IWorld 接口定义

```csharp
public interface IWorld
{
    // ========== 实体管理 ==========

    /// <summary>创建实体</summary>
    IECEntity CreateEntity();

    /// <summary>销毁实体</summary>
    void DestroyEntity(IECEntity entity);

    /// <summary>根据 ID 查找实体</summary>
    bool TryGetEntity(int entityId, out IECEntity entity);

    /// <summary>获取所有实体</summary>
    IReadOnlyList<IECEntity> GetAllEntities();

    // ========== 系统管理 ==========

    /// <summary>添加系统</summary>
    void AddSystem(IECSystem system);

    /// <summary>移除系统</summary>
    void RemoveSystem(IECSystem system);

    // ========== 查询 ==========

    /// <summary>创建查询</summary>
    IECQuery CreateQuery(IECMatcher matcher);

    /// <summary>获取匹配的所有实体</summary>
    IReadOnlyList<IECEntity> GetEntities(IECMatcher matcher);

    // ========== 生命周期 ==========

    /// <summary>初始化</summary>
    void Initialize();

    /// <summary>Tick</summary>
    void Tick(float deltaTime);

    /// <summary>销毁</summary>
    void Destroy();
}
```

## 4. World 的生命周期

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           World 生命周期                                   │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   ┌──────────┐     Initialize     ┌──────────┐     Tick     ┌──────────┐
│   │   New    │ ──────────────────▶│  Ready   │─────────────▶│  Running │
│   └──────────┘                   └──────────┘              └──────────┘
│                                                                  │
│                                                                  │ Destroy
│                                                                  ▼
│                                                          ┌──────────┐
│                                                          │ Destroyed│
│                                                          └──────────┘
│                                                                             │
│   状态说明：                                                               │
│   ├── New:      World 对象刚创建，尚未初始化                            │
│   ├── Ready:    已完成初始化，等待 Tick                                 │
│   ├── Running:  正在执行 Tick                                          │
│   └── Destroyed: 已销毁，不可再使用                                   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
