## 2. 管理器接口

```csharp
public interface IWorldManager
{
    /// <summary>创建世界</summary>
    IWorld Create(WorldCreateOptions options);

    /// <summary>销毁世界</summary>
    bool Destroy(WorldId id);

    /// <summary>尝试获取世界</summary>
    bool TryGet(WorldId id, out IWorld world);

    /// <summary>获取所有世界</summary>
    IReadOnlyList<IWorld> GetAll();

    /// <summary>Tick 所有世界</summary>
    void Tick(float deltaTime);
}
```
